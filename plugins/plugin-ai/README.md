# @baseplate-dev/plugin-ai

AI development agents plugin for Baseplate.

Generates AI agent configuration files (AGENTS.md, CLAUDE.md, .agents/) for your Baseplate projects, with setup instructions tailored to the coding agents your team uses.

## Features

- **AGENTS.md** — Project context, build commands, conventions for any AI agent
- **CLAUDE.md** — Claude Code-specific configuration (conditional)
- **.agents/baseplate.md** — Baseplate MCP server usage guide with per-agent setup commands

## Supported Agents

- Claude Code
- Cursor
- Gemini CLI
- GitHub Copilot (VS Code)

## Part of Baseplate Monorepo

This package is part of the Baseplate monorepo and extends Baseplate projects with AI agent configuration.
